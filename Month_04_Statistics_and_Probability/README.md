# Month 4: Statistics & Probability for Data Science

## Overview
Build a strong statistical foundation for data analysis and machine learning.

## Topics Covered
- Descriptive statistics and data distributions
- Probability theory and distributions
- Inferential statistics and hypothesis testing
- Regression analysis
- Statistical modeling techniques

## Key Libraries
- SciPy
- statsmodels

## Notebooks

| Notebook | Topic |
|----------|-------|
| [Week 1: Descriptive Statistics](Week_1_Descriptive_Statistics.ipynb) | Mean, median, variance, skewness, distributions |
| [Week 2: Probability Theory](Week_2_Probability_Theory.ipynb) | Probability rules, Bayes' theorem, common distributions |
| [Week 3: Inferential Statistics](Week_3_Inferential_Statistics.ipynb) | Hypothesis testing, t-tests, chi-square, ANOVA |
| [Week 4: Regression Analysis](Week_4_Regression_Analysis.ipynb) | Simple and multiple linear regression, OLS |
| [Week 5: Statistical Modeling](Week_5_Statistical_Modeling.ipynb) | Logistic regression, GLMs, model selection |
