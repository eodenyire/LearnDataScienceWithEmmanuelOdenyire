# Phase 0: Pre-Bootcamp Setup

**Duration:** 2 Weeks (Days 1–14)  
**Daily Commitment:** 15 hours/day  
**Total Hours:** 210 hours

## Overview

Phase 0 sets up everything you need to succeed in this bootcamp. You'll install your development environment, master Git and GitHub for version control, learn to use Google Colab, and develop the mindset of a data scientist.

## Setup Checklist

Before starting Phase 1, ensure you have:

- [ ] Python 3.8+ installed (or Google Colab account)
- [ ] Git installed and configured
- [ ] GitHub account created
- [ ] This repository forked and cloned
- [ ] `requirements.txt` dependencies installed
- [ ] `verify_setup.py` runs without errors
- [ ] First commit pushed to your fork
