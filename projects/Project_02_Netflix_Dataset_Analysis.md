# Project 02: Netflix Dataset Analysis

## Overview
Perform an end-to-end exploratory analysis of Netflix's content library to uncover trends and generate business insights.

## Dataset
[Netflix Movies and TV Shows – Kaggle](https://www.kaggle.com/shivamb/netflix-shows)

## Key Questions to Answer
1. How has Netflix's library grown from 2008 to 2021?
2. Which countries produce the most content?
3. What are the trending genres by year?
4. What is the typical duration of movies vs. TV shows?
5. Who are Netflix's most featured directors and actors?
6. Is there a seasonal pattern in when content is added?

## Technical Requirements
- Pandas for data wrangling
- Matplotlib and Seaborn for visualization
- Plotly for interactive charts (bonus)
- All visuals properly labeled and styled

## Deliverables
- Jupyter Notebook with narrative analysis
- 10+ visualizations
- Written insights section (300 words minimum)
- GitHub repository
