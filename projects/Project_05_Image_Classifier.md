# Project 05: Image Classifier

## Overview
Build a deep learning image classification model using CNNs and transfer learning.

## Dataset Options
1. **CIFAR-10** (built into Keras/PyTorch)
2. **Custom dataset**: collect 200+ images across 3+ categories
3. **Plant Disease** dataset (Kaggle)
4. **Waste Classification** dataset (Kaggle)

## Required Architectures
1. **Custom CNN** (built from scratch): at least 3 Conv2D layers
2. **Transfer Learning** (EfficientNet or ResNet50)

## Evaluation
- Training/validation/test curves
- Confusion matrix
- Top-5 misclassified images with analysis

## Bonus
- Deploy as a web app (Gradio or Streamlit)
- Apply Grad-CAM visualization

## Deliverables
- Training notebook
- Saved model files
- Deployed demo URL (bonus)
- GitHub repository
