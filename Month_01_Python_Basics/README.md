# Month 1: Python Basics for Data Science

## Overview
This month focuses on building a strong foundation in Python programming, essential for data science work.

## Topics Covered
- Introduction to Python programming
- Variables, data types, and operators
- Control flow, loops, and functions
- Data structures: lists, tuples, dictionaries, and sets
- String manipulation and error handling

## Notebooks
1. `Day_01_Introduction_to_Data_Science_and_Python.ipynb` - Getting started with Python
2. `Day_02_Setting_Up_Your_Python_Environment.ipynb` - Environment setup
3. `Day_03_Variables_and_Data_Types.ipynb` - Understanding variables and types
4. `Day_04_Operators_and_Expressions.ipynb` - Working with operators
5. `Day_05_Control_Flow.ipynb` - If statements and conditions

## Prerequisites
- Basic computer literacy
- Enthusiasm to learn!

## Getting Started
```bash
jupyter notebook Day_01_Introduction_to_Data_Science_and_Python.ipynb
```
