## How to Use

1. **Before attempting:** Complete all notebooks for the relevant week/phase
2. **Time yourself:** Respect the time limit to simulate exam conditions
3. **Score yourself:** Check the answer key at the bottom
4. **Review gaps:** For any wrong answers, return to the relevant notebook
5. **Retake:** If below passing score, review and retake after 24 hours

## Daily Self-Assessment Template

Create a daily habit of asking yourself:

1. What did I learn today? (3 key concepts)
2. What was confusing? (1–2 things to revisit)
3. Can I explain this to someone else? (The Feynman Technique)
4. What will I build with this knowledge?

## Scoring Guide

| Score | Level | Action |
|-------|-------|--------|
