# Month 02: Data Analysis with Pandas & NumPy

**Phase 1 | Month 2 | Bootcamp Days 29–56**  
**Daily Commitment:** 15 hours/day | **Total Hours:** 420 hours

## Overview

Month 2 takes you from Python programmer to practising data analyst. You will master NumPy — the numerical engine of all data science — and Pandas — the most widely used data manipulation library in the world. By the end of this month you will be able to clean, transform, join, and analyse real datasets.

## Learning Outcomes

By the end of Month 2 you will be able to:

1. Create, manipulate, and vectorise NumPy arrays across all dimensions
2. Apply linear algebra operations (matrix multiply, solve Ax=b, PCA) with NumPy
3. Load, clean, and transform real-world tabular datasets with Pandas
4. Filter, group, and aggregate data to answer business questions
5. Join multiple data sources using SQL-style merges
6. Handle time series data with resampling and rolling windows
7. Build a complete EDA pipeline from raw CSV to analytical insights
