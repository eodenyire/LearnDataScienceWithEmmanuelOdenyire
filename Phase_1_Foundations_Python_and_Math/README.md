# Phase 1: Foundations - Python & Mathematics for Data Science

**Duration:** 4 Months (Days 1–112 of the bootcamp)  
**Daily Commitment:** 15 hours/day  
**Total Hours:** 1,680 hours

## Overview

Phase 1 builds the essential programming and mathematical foundations every data scientist needs. By the end of this phase, you will be comfortable writing Python programs, manipulating data with NumPy and Pandas, creating visualizations, and applying statistics and probability to real-world problems.

## Learning Objectives

By the end of Phase 1, you will be able to:

- ✅ Write clean, Pythonic code with functions, OOP, and data structures
- ✅ Use NumPy for numerical computing and vectorized operations
- ✅ Manipulate datasets with Pandas (filtering, groupby, merging)
- ✅ Create compelling visualizations with Matplotlib, Seaborn, and Plotly
- ✅ Apply descriptive and inferential statistics
- ✅ Understand probability distributions and hypothesis testing
- ✅ Build a complete data analysis project from scratch
