# Month 01: Python Basics for Data Science

**Phase 1 | Month 1 | Bootcamp Days 1–28**  
**Daily Commitment:** 15 hours/day | **Total Hours:** 420 hours

## Overview

Month 1 takes you from zero to confident Python programmer. You will master all the Python concepts needed for data science: variables, control flow, functions, OOP, data structures, and more.

| Week | Theme | Days | Notebooks |
