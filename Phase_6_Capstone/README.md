# Phase 6: Capstone Projects (1 Month)

Demonstrate mastery by completing an end-to-end data science project that solves a real-world problem.

## Duration

## Learning Objectives
- Frame a real-world problem as a data science task
- Collect, clean, and explore data independently
- Build and evaluate appropriate models
- Deploy a working solution
- Present findings professionally
