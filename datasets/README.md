# Datasets

This directory contains datasets used throughout the bootcamp.

## Dataset Organization

Datasets are organized by topic or month:
- `sample_data/` - Small example datasets for learning
- `real_world/` - Larger, real-world datasets for projects
- `kaggle/` - Datasets from Kaggle competitions

## Usage Guidelines

1. **Sample Datasets**: Use for practice and exercises
2. **Citation**: Always cite the source of external datasets
3. **Privacy**: Never commit sensitive or personal data
4. **Size**: Keep datasets under 5MB when possible

## Common Datasets

### Included
- `sample_sales.csv` - Sample retail sales data
- `iris.csv` - Classic Iris flower dataset
- `titanic.csv` - Titanic passenger data

## Data Dictionary

Each dataset should have a corresponding data dictionary explaining:
- Column names and descriptions
- Data types
- Valid ranges
- Missing value indicators
- Source and date
