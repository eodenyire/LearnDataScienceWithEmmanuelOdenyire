# Month 10: Natural Language Processing (NLP)

## Overview
Learn to work with text data and build NLP applications.

## Topics Covered
- Text preprocessing techniques
- Tokenization and embedding methods
- Sentiment analysis and text classification
- Sequence-to-sequence models
- Introduction to transformers and BERT

## Key Libraries
- NLTK
- spaCy
- Transformers (Hugging Face)

## Notebooks

| Notebook | Topic |
|----------|-------|
| [Week 1: Text Preprocessing](Week_1_Text_Preprocessing.ipynb) | Cleaning, tokenization, stemming, lemmatization, TF-IDF |
| [Week 2: Tokenization and Embeddings](Week_2_Tokenization_and_Embeddings.ipynb) | Word2Vec, GloVe, t-SNE visualisation, TF-IDF classification |
