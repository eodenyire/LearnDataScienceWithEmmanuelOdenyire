# Phase 1: Python & Math (2 Months)

## Overview
Master Python programming fundamentals and the mathematical foundations of data science.

## Duration
**2 Months** | Months 1–2 + Month 4 (Statistics)

## Linked Modules
- [Month 01 – Python Basics](../Month_01_Python_Basics/)
- [Month 02 – Data Analysis with Pandas & NumPy](../Month_02_Data_Analysis_with_Pandas_and_NumPy/)
- [Month 04 – Statistics & Probability](../Month_04_Statistics_and_Probability/)

## Learning Objectives
- Write clean, Pythonic code with functions and OOP
- Use NumPy for numerical computing
- Apply Pandas for data manipulation
- Understand statistics, probability, and linear algebra fundamentals

## Weekly Topics

| Week | Topic |
|------|-------|

| 2 | Data Structures (lists, dicts, sets) |
| 3 | File Handling & Modules |
| 4 | Mini Data Analyzer Project |
| 5–8 | Pandas, NumPy, SQL |
| 9–12 | Statistics, Probability, Linear Algebra |
