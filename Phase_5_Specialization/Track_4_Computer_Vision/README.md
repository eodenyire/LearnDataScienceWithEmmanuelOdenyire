# Track 4: Computer Vision

## Overview
Apply deep learning to image and video understanding tasks.

## Learning Objectives
- Understand image data representation and augmentation
- Build and train CNNs for image classification
- Perform object detection with YOLO and SSD
- Apply transfer learning with pretrained models (ResNet, EfficientNet)
- Explore image segmentation
