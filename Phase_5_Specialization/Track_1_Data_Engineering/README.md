# Track 1: Data Engineering

## Overview
Learn to build scalable data pipelines and work with big data technologies.

## Learning Objectives
- Design and implement ETL/ELT pipelines
- Use Apache Kafka for real-time streaming
- Process large datasets with Apache Spark
- Work with data warehouses (BigQuery, Redshift)
- Orchestrate workflows with Apache Airflow

## Notebooks
- [Data Pipelines with Kafka](data_pipelines_with_kafka.ipynb)
- [Spark Data Processing](spark_data_processing.ipynb)

## Tools & Technologies
- Apache Kafka, Apache Spark, Apache Airflow
- PostgreSQL, BigQuery
- Docker, Kubernetes (basics)
- dbt (data build tool)
