# Track 3: Natural Language Processing

## Overview
Master NLP from text preprocessing to large language models and chatbot development.

## Learning Objectives
- Preprocess and represent text data (TF-IDF, word embeddings)
- Fine-tune transformer models (BERT, GPT)
- Build question-answering and summarization systems
- Develop conversational AI / chatbots
- Understand prompt engineering

- [Build a Chatbot/QA System](build_chatbot_qa_system.ipynb)

- spaCy, NLTK
- LangChain
