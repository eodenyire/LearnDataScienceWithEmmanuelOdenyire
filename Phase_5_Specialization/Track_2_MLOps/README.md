# Track 2: MLOps

## Overview
Bridge the gap between ML experimentation and production deployment.

## Learning Objectives
- Track experiments with MLflow
- Containerize models with Docker
- Build REST APIs for model serving (FastAPI/Flask)
- Set up CI/CD pipelines for ML models
- Monitor model performance in production

## Notebooks
- [Docker Model Deployment](docker_model_deployment.ipynb)
- [MLflow Experiment Tracking](mlflow_experiment_tracking.ipynb)

## Tools & Technologies
- MLflow, DVC, Weights & Biases
- Docker, FastAPI
- GitHub Actions (CI/CD)
- Prometheus & Grafana (monitoring)
