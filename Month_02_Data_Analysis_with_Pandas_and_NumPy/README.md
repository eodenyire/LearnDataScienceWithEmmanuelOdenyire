# Month 2: Data Analysis with Pandas & NumPy

## Overview
Learn to work with numerical data using NumPy and data manipulation with Pandas.

## Topics Covered
- Introduction to NumPy arrays and operations
- Advanced NumPy features
- Pandas Series and DataFrames
- Reading and writing data with Pandas
- Data cleaning and transformation techniques

## Prerequisites
- Completion of Month 1 or equivalent Python knowledge

## Key Libraries
- NumPy
- Pandas
