# Phase 2: Data Science Core — Cleaning, EDA & Machine Learning

**Duration:** 4 Months (Days 113–224 of the bootcamp)  
**Daily Commitment:** 15 hours/day  
**Total Hours:** 1,680 hours

## Overview

Phase 2 transforms you from a Python programmer into a practicing data scientist. You will learn to clean messy real-world data, conduct thorough exploratory analyses, and build your first machine learning models. This phase bridges foundational skills with cutting-edge ML.

## Phase Structure

| Month | Topic | Weeks | Days |

## Learning Objectives

By the end of Phase 2, you will be able to:

- ✅ Clean and preprocess real-world messy datasets
- ✅ Conduct systematic EDA and generate insights
- ✅ Build and evaluate regression and classification models
- ✅ Apply clustering and dimensionality reduction
- ✅ Use scikit-learn's full ML pipeline
- ✅ Apply ensemble methods (Random Forest, XGBoost)
- ✅ Tune hyperparameters with cross-validation
- ✅ Explain model predictions with SHAP values
