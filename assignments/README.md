## Submission Guidelines

### Format
- Submit as a Jupyter Notebook (`.ipynb` file)
- Filename: `PhaseX_AssignmentName_[YourStudentID].ipynb`
- Or share a Colab link (with edit access to instructor)

### Requirements
1. **Title cell:** Your name, student ID, date, assignment name
2. **Introduction:** Brief description of the problem
3. **Approach:** Markdown explaining your methodology
4. **Code:** Clean, documented code in cells
5. **Results:** Clear outputs and visualizations
6. **Conclusion:** What you learned, limitations, next steps

### Marking Rubric
| Criterion | Weight |
|-----------|--------|
| Code correctness | 40% |
| Code quality & documentation | 20% |
| Approach & problem-solving | 20% |
| Results & visualization | 10% |
| Conclusions | 10% |

## Late Submission Policy

- Same day, within 3 hours: No penalty
- Next day: -20%
- After 2 days: -50%
- After 1 week: Not accepted (contact instructor)

## Academic Integrity

All assignments must be your own work. You may:
- Use the provided notebooks as reference
- Use documentation and textbooks
- Discuss concepts with peers

You may NOT:
- Copy code from other students
- Submit AI-generated solutions without understanding them
- Share your completed solution before the deadline
