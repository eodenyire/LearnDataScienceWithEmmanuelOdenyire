# Phase 0 Assignment: My Data Science Journey Plan

## Objective
Reflect on your goals, create a structured learning plan, and set up your development environment.

## Tasks

### Part 1: Personal Reflection (200–300 words)
Answer the following questions:
1. Why do you want to learn data science?
2. What problems do you hope to solve with data science?
3. What is your current technical background?
4. What does success look like for you after this bootcamp?

### Part 2: Environment Setup Checklist
Complete and screenshot each item:
- [ ] Python 3.8+ installed (run `python --version`)
- [ ] Jupyter Notebook or JupyterLab installed
- [ ] VS Code with Python and Jupyter extensions
- [ ] Git installed and configured (`git config --global user.name/email`)
- [ ] GitHub account created
- [ ] Google Colab account setup
- [ ] `requirements.txt` installed from this repository

### Part 3: GitHub Portfolio Repository
1. Create a new GitHub repository called `ds-bootcamp-[your-name]`.
2. Add a `README.md` with:
   - Your name and background
   - Learning goals for the bootcamp
   - Technologies you'll use
3. Push the repository to GitHub.

### Part 4: 12-Month Study Plan
Create a table or calendar showing:
- Weekly study hours
- Which Phase/Month you'll complete each week
- Milestones (project deadlines, quiz dates)

## Submission
- GitHub repository link
- Answers to Part 1 (in your README or a separate file)
- Screenshot of your working Jupyter environment
