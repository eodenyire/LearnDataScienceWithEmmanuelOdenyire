# Phase 6 Final Capstone Project

## Overview
The capstone project is your opportunity to demonstrate end-to-end data science skills on a real-world problem of your choosing. This is your portfolio centerpiece.

## Project Requirements

### 1. Problem Statement
- Clearly define the business or social problem
- Explain why data science is the right tool
- Define measurable success criteria

### 2. Data Collection
Choose one or more:
- Kaggle datasets (properly cited)
- Public API data collection
- Web scraping (ethically, with permissions)
- Synthetic data for simulation

Minimum: **1,000+ rows, 5+ meaningful features**

### 3. Exploratory Data Analysis
- Comprehensive EDA with 10+ visualizations
- Statistical analysis of key variables
- Hypothesis generation and testing

### 4. Machine Learning / Deep Learning
- At least 3 different algorithms compared
- Proper train/validation/test split (no data leakage)
- Cross-validation for model selection
- Hyperparameter tuning with documented results
- Feature importance analysis

### 5. Deployment (Required)
Deploy as one of:
- **Streamlit app** hosted on Streamlit Cloud
- **FastAPI** Docker container
- **Gradio** demo on HuggingFace Spaces

### 6. Presentation
- 10-15 minute presentation to cohort
- Non-technical executive summary (1 page)
- Technical blog post (Medium, dev.to, or personal site)
