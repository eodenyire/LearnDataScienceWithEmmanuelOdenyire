# Phase 2 Assignment: Netflix Dataset Analysis

## Objective
Perform a comprehensive exploratory data analysis on the Netflix titles dataset and tell a compelling data story.

## Dataset
Download from Kaggle: [Netflix Movies and TV Shows](https://www.kaggle.com/shivamb/netflix-shows)
File: `netflix_titles.csv`

## Analysis Requirements

### Part 1: Data Cleaning (20 points)
- Handle missing values (strategy must be justified)
- Parse and clean the `date_added` and `duration` columns
- Standardize `listed_in` (genres) column
- Document all cleaning decisions

### Part 2: Exploratory Analysis (40 points)
Answer these questions with visualizations:
1. How has Netflix's content library grown over the years?
2. What is the ratio of Movies vs. TV Shows? Has it changed over time?
3. Which countries produce the most Netflix content?
4. What are the most popular genres? Are there genre trends over time?
5. What is the distribution of movie durations and TV show seasons?
6. Which directors and actors appear most frequently?

### Part 3: Insights Report (40 points)
Write a 500-word narrative answering:
- What surprised you most about the data?
- What content strategy can you infer Netflix has been following?
- What are the limitations of this dataset?

## Deliverables
- Jupyter Notebook with clean, narrative-style analysis
- All visualizations properly labeled (title, axes, legend)
- Markdown cells explaining each section
