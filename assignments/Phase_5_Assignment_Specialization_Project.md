# Phase 5 Assignment: Specialization Track Project

## Objective
Demonstrate mastery of your chosen specialization track by completing a focused project.

## Track-Specific Requirements

### Track 1: Data Engineering
Build an end-to-end data pipeline:
- Source: public API or synthetic data generator
- Ingestion: Kafka producer streaming data
- Processing: Spark or pandas transformation
- Storage: PostgreSQL or Parquet files
- Orchestration: Airflow DAG (optional)
- Dashboard: Streamlit or Grafana (optional)

### Track 2: MLOps
Deploy a machine learning model to production:
- Train a model on any dataset
- Package as a FastAPI service
- Containerize with Docker
- Track experiments with MLflow
- CI/CD pipeline with GitHub Actions
- Monitor predictions (logging + alerts)

### Track 3: Natural Language Processing
Build an NLP application:
- Fine-tune a HuggingFace transformer on custom data
- Build a document Q&A system using RAG + LangChain
- Deploy as a Streamlit or Gradio app
- Evaluate with BLEU, ROUGE, or task-specific metrics

### Track 4: Computer Vision
Build a computer vision application:
- Train an object detector with YOLOv8 on custom data (100+ images)
- Apply transfer learning for multi-class classification
- Deploy with OpenCV + FastAPI
- Evaluate with mAP and confusion matrix

## Common Requirements (All Tracks)
- GitHub repository with clear README
- Architecture diagram
- 10-minute demo video (Loom or similar)
- Written reflection: What worked? What would you do differently?
